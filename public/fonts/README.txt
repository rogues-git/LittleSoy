Place the Royal Hefana font file here.

    ROYALHefanaRegular.ttf

Full path:
    public/fonts/ROYALHefanaRegular.ttf

This is referenced by the @font-face rule in src/app/globals.css
(font-family: "Royal Hefana") and applied via the Tailwind class
`font-royal-hefana` wherever the brand name "Little Soe" appears.

If you use a different file name or format (.otf / .woff2), update the
`src: url(...)` line in src/app/globals.css to match.
